## When to use

- You asked "audit my site", "improve my SEO", "why isn't this page ranking".
- You want an action plan, not a 60-page PDF.

## What it does

Looks at your URL and returns a prioritized report:

- **🔴 Blockers**: accidental noindex, broken redirects, wrong canonical, robots blocking.
- **🟡 High impact**: duplicate title, missing H1, missing schema, weak internal linking, critical page speed.
- **🟢 Marginal**: alt-text, weak meta descriptions, incomplete sitemap.xml.

Each item includes: the problem, the exact fix, and time-to-fix (5 min / 1 hour / 1 day).

## How to use

1. Pass the URL (or several if comparing).
2. Add context if useful: *"we compete with X.com, want to rank for 'term Y'."*
3. The skill returns the prioritized list and you work top-down.

## Best for

Founders doing their own SEO, marketing teams who want a second opinion before paying an agency, agencies producing baseline reports for new clients.
