# TerminalSync — Brand Assets

Logos listos para usar en producto, redes sociales y materiales impresos.

| Archivo | Uso recomendado | Fondo |
|---|---|---|
| `logo-round.svg` | Perfil de Instagram, Facebook, X, LinkedIn, Discord | Cualquiera (el círculo es el marco) |
| `logo-square.svg` | App icon, favicon alternativo, banners | Cualquiera |

Los SVG son **1024×1024**, vectoriales, escalan sin pérdida.

## Cómo exportar a PNG (si la red pide JPG/PNG)

macOS:
```bash
sips -s format png public/brand/logo-round.svg --out ~/Desktop/logo-round-1024.png
```

O con Figma / Sketch: importá el SVG, Export → PNG 2x.

## Paleta

- Azul primario: `#3b82f6 → #0284c7` (gradient)
- Chip oscuro: `#0b1120`
- Chip claro: `#ffffff`
- Chip rosé: `#f5b391`

Los tres chips representan los tres temas de terminal del producto (dark classic / white clean / rosé warm).
